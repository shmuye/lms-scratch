# Full Stack Library Management System

## Features

---

- Full Authentication flow (Register, Login, Logout, deleteAccount, change password)

- Role based Access Control with Roles

    - ADMIN

    - Librarian

    - Reader ( normal user)

- Each with their own permissions

- Unique dashboard for each user

- Readers can browse books, borrow, return books, 

- Librarians control the status of books

## Tech Stack

---

### Frontend

- **React** - The main framework for the frontend
 
- **Typescript** - type safe programming language

- **Redux toolkit** - for state management

- **React router dom** - for routing

- **Lucide react**  - for icons

### Backend

- **Express.js** - Node.js backend framework

- **javascript** - programming language

- **Multer** - file upload and storage
