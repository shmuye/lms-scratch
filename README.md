# Library Management System

- A full stack library management system build using MERN(React, Node, Express, Mongodb) with full authentication and autherization flow.

## Features

- Browsing ans searching book functionality
- Admin dashboard and librarian dashboard
- full Authentication system using jwt
- state management using Redux Toolkit
- server statemanagement using Tanstack query

## Tech Stack

### Backend

- Node.js
- Express.js
- Mongodb
- Render for sending emails
- jwt

### Frontend

- React.js
- Redux - for state management
- Tanstack query - to effiently manage server state

## Setup Instructions

### 1. Clone the repository

```bash
git clone <repository-url>
cd library-management-system

cd backend
npm install
npm run dev

cd ../frontend
npm install
npm run dev
```
